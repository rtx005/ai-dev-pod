# AI Dev Pod

AI-powered virtual software development team.

Agents included:
- Business Analyst
- Design Agent
- Developer Agent
- Testing Agent
- Project Manager

## Tech Stack

Frontend:
HTML, CSS, JavaScript

AI Engine:
Ollama + Llama3

## Run Locally

Start Ollama:

ollama serve

Start web server:

python3 -m http.server 8001

Open:

http://localhost:8001
